Intro

Hey!

My name's Logan and I'm a U3 Computer Science - Artificial Intelligence student at McGill University in Montreal, QC. I'm extremely interested in both the theoretical and practical aspects of both AI and Software Development, and I'm always looking for new opportunities to learn and grow as a developer and researcher. I'm especially passionate about the applications of AI in the fields of healthcare and education. Feel free to look at my class projects (under "Education") and side projects (under "Projects") for a more specific idea of what I've been working on recently, or "About" to learn more about me and my personal life. If you'd like to get in touch, feel free to send me an email or connect with me on LinkedIn (both under "Links")!

Projects

Here are some of the projects I've worked on recently:

  • In COMP 551 I completed four team projects, implementing a variety of ML models: linear regression, logistic regression, MLPs, CNNs, Naive Bayes, and BERT classification. Our group also wrote reports on our findings. The code and report for each of these projects is available upon request.
  • In COMP 424, my partner and I created an agent, "Minimax Search with Alpha-Beta Pruning and Fixed Breadth and Depth", available on my GitHub, which plays the game "Colosseum Survival!". The agent won 63.78% of its games against agents created by other students in the class and 100% of its games against a random agent, finishing in the 73rd percentile of the class. The report we wrote about its performance and our design decisions is available upon request.
  • Using JavaScript, I created a chrome extension, Tabulator, which allows users to quickly group tabs by URL using its simple, single-button interface. It is currently available on the Chrome Web Store, with over six users and a 5-star rating.
  • In Jan 2022, I participated in the McHacks 9 Hackathon, where my team and I created a web application, ARoom (Devpost available here), which allowed users to create and join voice chat rooms with people nearby. We receieved the Huawei Custom Challenge: Best Pandemic Life-Related Hack award for our project.
  • For the class COMP 322, I created a local version control system using C++, allowing users to save and track changes between versions of a file through the command line. A demo is available upon request.
  • I'm currently working on a few projects:
    • A reinforcement learning agent, trained using Deep Q-Learning to maximize the agent's earnings in a game of Blackjack with varying shoe sizes, deck penetration, and numbers of players.
    • A data visualization project, using a custom data set I collected and cleaned from the book Red Rising, which shows a social network of the characters in the book (and their relationships with each other).
    • A desktop application in Python allowing users to track the amount of time they spend studying each subject per day, allowing them to allocate their time more effectively and efficiently.

    For a complete list of personal projects, please view my GitHub profile.
